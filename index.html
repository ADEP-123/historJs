Historia Js

4 de Diciembre 1995
Brendan Eich de Netscape crea JavaScript en 10 dias con el nombre de "Mocha", posteriormente renombrado a "LiveScript" y finalmente a "JavaScript", se cree que como estrategia publicitaria por la popularidad del lenguaje JAVA

Finales de Diciembre 1995
Se introduce JavaScript en el navegador Netscape Navigator 2.0 como un lenguaje de secuencias de comandos del lado del cliente. La idea era permitir a los desarrolladores web crear páginas web interactivas y dinámicas.

Inicios de 1997
JavaScript es estandarizado por ECMA International bajo el nombre de ECMAScript. Esto lleva a la creación de la primera versión estándar de JavaScript, conocida como ECMAScript 1. Desde entonces, ha habido varias versiones de ECMAScript, cada una introduciendo nuevas características y funcionalidades al lenguaje.

Actualidad
JavaScript se ha convertido en uno de los lenguajes de programación más populares en internet y más usados. Al principio, muchos desarrolladores renegaban del lenguaje porque el público al que va dirigido lo formaban aficionados, entreotras razones.La llegada de AJAX devolvió JavaScript a la fama y atrajo la atención de muchos otros programadores. Como resultado de esto hubo una proliferación de un conjunto de frameworks y bibliotecas de ámbito general, mejorando las prácticas de programación con JavaScript, y aumentado el uso del mismo fuera de los navegadores web, como se ha visto con la proliferación de entornos JavaScript del lado del servidor
